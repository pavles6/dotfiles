### A collection of my dotfiles for mac and linux machines

uses [chezmoi](https://www.chezmoi.io/) to handle and manage os specific configuration

Requires `chezmoi`, `homebrew` (on mac systems) or `pacman` + `yay`/`paru` (on linux systems)

List of packages that will be automatically installed via `brew` or `pacman`/`paru` is located [here](.chezmoidata/packages.yml)
